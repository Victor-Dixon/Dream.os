**ROI Formula:**
```
Value = (
    50 (if has_readme) +
    50 (if has_tests) +
    30 (if has_license) +
    40 (if has_ci_cd) +
    uniqueness_score * 3 +
    stars * 20 +
    commits * 2
)

Effort = (
    1 (base) +
    file_count * 0.5 +
    20 (if no tests) +
    10 (if no license) +
    15 (if no ci_cd)
)

ROI = Value / Effort
```
