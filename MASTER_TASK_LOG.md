# Master Task Log

This log tracks all site audit, information architecture, copy, and route tasks across all managed websites.

## Task Format
- `[ ]` = Uncompleted
- `[x]` = Completed
- Format: `- [ ] [CATEGORY][PRIORITY][TASK-ID] description`

## Categories
- `SITE_AUDIT` - Site audit findings (broken links, errors, etc.)
- `SITE_IA` - Information architecture tasks
- `SITE_COPY` - Copy/content tasks
- `SITE_ROUTES` - Route/URL structure tasks
- `SITE_SEO` - SEO optimization tasks
- `SITE_PERF` - Performance optimization tasks

## Priorities
- `CRITICAL` - Blocks functionality or security issues
- `HIGH` - Important user-facing issues
- `MEDIUM` - Nice-to-have improvements
- `LOW` - Minor optimizations
