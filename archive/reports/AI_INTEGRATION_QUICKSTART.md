## üö® **Common Integration Mistakes (Avoid)**

### ‚ùå **Don't**: Use AI for everything
```python
# Bad: AI for simple calculations

# Good: AI for complex analysis
result = engine.reason("Analyze system performance bottlenecks", mode="technical")
```

### ‚ùå **Don't**: Ignore context
```python
# Bad: Generic queries

# Good: Specific, contextual queries

```

```python

```
