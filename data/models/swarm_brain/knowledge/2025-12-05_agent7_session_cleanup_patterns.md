## üìã Session Cleanup Checklist

### **1. Create/Update passdown.json**
- Document current state, blockers, next steps
- Include deliverables, achievements, lessons learned
- Set clear handoff notes for next session
- **Impact**: Prevents context loss between sessions

### **2. Post Devlog to Discord**
- Create comprehensive devlog entry
- Document session accomplishments
- Include metrics, achievements, next actions
- **Impact**: Preserves session history for reference

### **3. Update Swarm Brain**
- Share learnings and patterns
- Document best practices discovered
- Contribute to collective knowledge
- **Impact**: Enables swarm learning and pattern reuse

### **4. Clean Workspace**
- Organize inbox (archive old messages)
- Remove temporary files
- Organize workspace structure
- **Impact**: Improves efficiency for next session

### **5. Update status.json**
- Mark session complete
- Set status to "READY_FOR_NEW_ASSIGNMENTS"
- Update next actions
- **Impact**: Clear status for Captain and other agents

## üí° Key Learnings

### **Session Cleanup Benefits**
1. **Smooth Handoffs**: Clear documentation prevents context loss
2. **Knowledge Preservation**: Swarm brain updates preserve collective learning
3. **Workspace Efficiency**: Clean workspace improves productivity
4. **Status Clarity**: Updated status files enable better coordination
5. **Pattern Reuse**: Documented patterns benefit future sessions

### **Best Practices**
- **Complete All Items**: Don't skip cleanup steps
- **Be Comprehensive**: Document thoroughly for future reference
- **Update Swarm Brain**: Share learnings with the swarm
- **Organize Workspace**: Remove clutter, maintain structure
- **Clear Status**: Set explicit status for next session
