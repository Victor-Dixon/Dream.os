## üé• PRODUCTION NOTES

### **Recording Setup**
1. **Screen Recording Software**: 
   - OBS Studio (free) or ScreenFlow (Mac) or Camtasia
   - 1920x1080 resolution
   - 60fps for smooth animations

2. **Discord Setup**:
   - Use test server with WeAreSwarm bot
   - Clean, professional channel names
   - Set up realistic agent status scenarios

3. **Audio**:
   - Use good microphone (Blue Yeti, Rode, or similar)
   - Record in quiet room
   - Use noise reduction in post

4. **Lighting** (if showing face):
   - Natural light or ring light
   - Face the camera, good eye contact

### **Post-Production**
1. **Editing Software**: 
   - DaVinci Resolve (free) or Premiere Pro
   - Add text overlays
   - Smooth transitions
   - Color correction

2. **Music**:
   - Use royalty-free music (Epidemic Sound, Artlist, YouTube Audio Library)
   - Upbeat, tech-focused, not distracting
   - Keep volume low (20-30% of voiceover)

3. **Text Overlays**:
   - Consistent font (Inter, Poppins, or similar)
   - Bold for emphasis
   - White/light color for visibility
   - Subtle animations (fade in, slide)

4. **Length**:
   - Target: 60-90 seconds
   - Cut ruthlessly - every second counts
   - Faster pace = more engagement
